# 국제학교를 선택한다는 건 - 인터랙티브 에세이 북 (Godot Engine)

인터랙티브 에세이 북 **"국제학교를 선택한다는 건"**의 Godot Engine 버전 프로젝트입니다.
학부모이자 교직원의 시선으로 담아낸 국제학교 선택에 대한 사색과 성찰을 인터랙티브한 연출(비, 눈, 구름, 터치 반응형 비행기 등)과 함께 읽을 수 있는 멀티플랫폼 전자책 앱입니다.

---

## 🚀 고도 엔진(Godot Engine) 초보자를 위한 설치 및 실행 가이드

고도 엔진을 처음 접하시는 분도 3분 만에 설치하고 이 프로젝트를 실행할 수 있습니다. 차근차근 따라 해 보세요!

### 1단계: 고도 엔진 다운로드하기
고도 엔진은 설치 프로그램(인스톨러)이 필요 없는 아주 가볍고 안전한 프로그램입니다.
1. [고도 엔진 공식 다운로드 페이지(Godot Download)](https://godotengine.org/download)에 접속합니다.
2. **Godot Engine - Standard version** (또는 Windows용 Standard) 버튼을 눌러 다운로드합니다.
3. 다운로드된 ZIP 압축 파일을 풀면 나오는 **`Godot_v4.x.x-stable_win64.exe`** 파일이 바로 실행 프로그램입니다. (더블 클릭하면 바로 실행됩니다.)

### 2단계: 프로젝트 파일 준비하기
1. GitHub에서 이 프로젝트를 ZIP 파일로 다운로드했다면 압축을 해제합니다. (또는 Git을 사용해 폴더를 복제합니다.)
2. 폴더 내부에 `project.godot` 파일이 들어있는지 확인합니다.

### 3단계: 고도 엔진에 프로젝트 불러오기
1. 1단계에서 압축을 푼 **고도 엔진 실행 파일(.exe)**을 더블 클릭해 켭니다.
2. 처음에 나타나는 프로젝트 관리자(Project Manager) 창 우측 메뉴에서 **`Import` (가져오기)** 버튼을 클릭합니다.
3. 경로 탐색기 창이 뜨면, 2단계에서 준비한 프로젝트 폴더 내부로 들어가 **`project.godot`** 파일을 선택하고 `Open`을 누릅니다.
4. **`Import & Edit` (가져오기 및 편집)** 버튼을 클릭하면 에디터 창이 열립니다.

### 4단계: 에세이 북 실행하고 플레이하기
1. 에디터 창이 열리면 화면 오른쪽 맨 위 모퉁이에 있는 **▶ 모양의 Play 버튼** (또는 단축키 **`F5`**)을 누릅니다.
2. 게임 화면이 뜨면 감성적인 인트로 연출과 구름, 비 효과가 나타납니다.
3. **[읽기 시작하기 ▶]** 버튼을 누르면 본문으로 들어갑니다.
4. 책 본문 화면을 **마우스로 클릭하거나 화면을 탭**해 보세요! 예쁜 터치 스파클 효과와 함께 배경의 종이비행기가 클릭한 곳으로 날아와 360도 공중제비를 도는 인터랙티브 연출을 감상하실 수 있습니다.

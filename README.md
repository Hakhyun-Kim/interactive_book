# 국제학교를 선택한다는 건 - 인터랙티브 에세이 북 (Godot Engine)

인터랙티브 에세이 북 **"국제학교를 선택한다는 건"**의 Godot Engine 버전 프로젝트입니다.
학부모이자 교직원의 시선으로 담아낸 국제학교 선택에 대한 사색과 성찰을 인터랙티브한 연출(비, 눈, 구름, 터치 반응형 비행기 등)과 함께 읽을 수 있는 멀티플랫폼 전자책 앱입니다.

## 주요 특징
- **감성적인 화면 연출**: 각 챕터별 분위기에 맞는 실시간 백그라운드 효과 (비, 별무리, 아지랑이, 구름 등)
- **인터랙티브 요소**: 마우스나 터치에 반응하여 날아다니고 공중제비를 도는 귀여운 종이비행기 연출
- **반응형 본문 뷰**: 모바일/태블릿 규격에 대응하는 종이 재질 느낌의 UI 레이아웃
- **성찰 카드**: 챕터별 사색 노트를 강조하여 표시하는 세련된 UI 카드 디자인
- **최신 Godot 4 기반**: 경량 및 고성능 구동을 보장하는 GDScript 기반 구현

## 실행 및 편집 방법
1. Godot Engine (v4.7 이상 권장)을 실행합니다.
2. `Import`를 누르고 본 프로젝트 루트의 `project.godot` 파일을 선택하여 가져옵니다.
3. `F5` 키를 눌러 프로젝트를 실행할 수 있습니다.
